VINTAGE WEBPOST

A creative web tool to design and send personalized digital postcards.
This project is an interactive website that allows users to create beautifully designed digital postcards based on simple user inputs. With an intuitive and minimalist interface, the site invites users to:

  • Upload or select an image for the main postcard visual
  • Choose a stamp image to enhance the look and feel
  • Select an occasion (e.g., Birthday, Anniversary, Graduation, Get well soon, Christmas)
  • Enter recipient details (relationship)
  • Pick a date to personalize the message even more
Once all fields are filled, the site automatically generates a shareable postcard that combines all elements into a visually appealing layout. You can also download the postcard by Command P and Save as PDF.
Built with HTML, CSS, and JavaScript, the project focuses on user-friendly interaction and aesthetic design.
